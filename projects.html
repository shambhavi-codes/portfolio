MMM Media Data Pipeline — SQL

Engineered a SQL data wrangling pipeline to clean, standardize, and consolidate weekly media data across 3 drivers — Paid Social (Facebook & Instagram), Digital Video (YouTube), and Paid Search (Google Search) — into a single unified dataset using UNION ALL and reusable SQL views. The pipeline ingests raw platform CSV exports, resolves inconsistent column structures across sources, and delivers 416 rows of analysis-ready data covering 104 weeks (2 years), reducing data preparation time and enabling the modeling team to access clean data without manual query rebuilding. All data used is anonymized; the pipeline replicates the logic and workflow of real MMM data preparation without exposing any client information.
